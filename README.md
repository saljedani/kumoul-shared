# Kumoul Cloud File Sharing System

## Overview
This project is a cloud-based file sharing system built for Kumoul Engineering. It enables uploading, storing, and managing files using AWS S3 with a scalable backend architecture.

## Architecture
- AWS S3 for file storage
- Flask backend for handling requests
- AWS infrastructure for scalability

## Features
- Upload files to S3
- Manage stored files
- Secure file handling
- Scalable backend system

## Tech Stack
- Python (Flask)
- AWS S3
- EC2 (if used)
- Terraform (if used)
